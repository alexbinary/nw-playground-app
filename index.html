NW.js Playground

Curious about what you can do with NW.js and how you can do it ? Don't want to setup a project just to test one feature or one line of code ? You've come to the right place !

NW.js Playground lets you write and execute pieces of code without having to setup a NW.js app. Just type your code below and click Run. 


	var dare = 'try me';
	console.log(dare);  // Open inspector to see the result
